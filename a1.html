Assignment 1: Blinking LEDs

  // Arduino sketch: staggered LED blinks by tick
  
  int LED1 = 13;
  int LED2 = 8;
  int LED3 = 4; 
  // make all LED pin# a variable
  
  int tick = 0;    // set a tick, that makes LEDs blink at different ticks
  int blink = 50;  // blink duration (ms)
  
  void setup() {
    pinMode(LED1, OUTPUT);
    pinMode(LED2, OUTPUT);
    pinMode(LED3, OUTPUT);
    // set LED pins to OUTPUT
  
    digitalWrite(LED1, LOW);
    digitalWrite(LED2, LOW);
    digitalWrite(LED3, LOW);
    // set LEDs to off
  }
  
  void loop() {
    // the delay between ticks
    delay(450);
    tick++;
  
    // for each tick, LED1 blinks
    digitalWrite(LED1, HIGH);
  
    // for every two ticks, LED2 blinks
    if (tick % 2 == 0) {
      digitalWrite(LED2, HIGH);
    }
  
    // for every four ticks, LED3 blinks
    if (tick % 4 == 0) {
      digitalWrite(LED3, HIGH);
    }
  
    delay(blink);
  
    // turn off LEDs after blink duration
    digitalWrite(LED1, LOW);
    digitalWrite(LED2, LOW);
    digitalWrite(LED3, LOW);
  }

Resistor Calculation

DC forward current (design): 20 mA = 0.02 A

Typical red LED forward voltage: 1.85 V

Required R at 5 V: R = (5 − 1.85) / 0.02 = 157.5 Ω

Chosen value: 220 Ω (safer, ~14.3 mA). Resistor power ≈ 0.045 W → 1/8 W or 1/4 W ok.

Additional Questions

How many LEDs could you blink independently? How much current would that draw?

Up to 18 LEDs (avoid D0/D1). 330 Ω → ~9 mA each, ~163 mA total. 220 Ω → ~13.6 mA each; cap ~14 LEDs (~191 mA total).

How fast do you need to blink your LEDs until you no longer can tell that they are blinking?

About 60 Hz (period ~16 ms) looks continuous.
